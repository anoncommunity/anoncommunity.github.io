ELECTRUM - NETWORK SERVER

Here you can enter the details of the server - the current servers are shown in the window and these only need updated if you have something different showing up in your wallet - remember to also update the port

To change servers, uncheck the 'Select server automatically' checkbox and simply enter the details of one of the servers as shown on the right. Yoou may need to restart the wallet for this change to take effect

ELECTRUM - NETWORK PROXY

Make sure that the proxy box is unchecked

After updating to use the published servers you should restart the wallet to ensure it's using the new settings
