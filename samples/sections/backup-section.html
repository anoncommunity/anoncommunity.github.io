WALLET - BACKUP

We shouldn't really have to stress the importance of backup when it comes to something as important as crypto wallets - but you MUST back up your wallet.dat file and export your keys to a secure envronment. Remember it's your investment - so invest a little time to ensure it's secure.

We've made this a simple procedure but it's down to you to make sure that everything goes as planned - so not only do you backup as directed, you also MUST ensure that you can read the exported keys. And we advise at least two separate backup locations which should be chosen so that it's extremely unlikely that both sets of backups could be destroyed at any one time.

To backup your wallet to a file ensure that the exportdir option is set in the zice.conf file (see 'export directory' previously)

Once that's complete simply press the 'backup' button above and this should make up a backup of your wallet to the export directory configured under the export dir section

export keys

To backup your private keys to a file ensure that the exportdir option is set in the zice.conf file (see 'export directory' previously)

Once that's complete simply press the 'export keys' button above and this should export your private keys to the export directory configured under the export dir section
